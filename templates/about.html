About This Project 🌿

The Smart Crop Monitoring System is an IoT-enabled platform that integrates real-time sensor data, machine learning crop predictions, and SHAP-based analysis to provide intelligent agricultural decisions. It supports farmers in selecting the right crops based on real-time environmental data such as temperature, humidity, soil moisture, and pH.

🌡️ Real-Time Monitoring

Continuously fetches data from sensors using ThingSpeak API to monitor field conditions.

🤖 Crop Prediction

Utilizes a trained Random Forest model to predict the best crop based on current conditions.

📋 Actionable Guidelines

Provides recommendations to improve field parameters for better crop yield and health.

🔍 Explainability with SHAP

Explains why a crop was recommended using SHAP values and helps fine-tune farming strategies.
